

# How it's done in other languages

## C

```
<data_type> <name>'['<array_size>']'
int data[100];

```

## Rust
```
let data: [i32; 100];
let data: [i32; 100] = [0; 100];
```

## Swift
```
var data = [Int](repeating: 0, count 100)
var data = Array(repeating: 0, count: 100)
```

## Go
```
var data [100]int
```

## Odin
data := [100]int{}
